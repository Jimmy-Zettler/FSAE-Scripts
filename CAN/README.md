Scripts for analyzing CAN buses

Typically use a PEAK Systems PCAN-USB tool for reading data, make sure to use the optocoupler version if working on HV systems

Favorite CAN Softwares:
  SavvyCAN - Great open source, lets you see frames per second and export data as .csv
  PCAN-view - Super easy to use for reading and sending messages, doesn't let you use DBC or export data easily though
